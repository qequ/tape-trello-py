# Tape-o-Trello

A little API to exchange information between your Trello Boards and [Tape]{https://aeriform.itch.io/tape}.2

# Usage

```
tt = TrelloToTape(api_key='your-key', api_secret='your-secret')
tt.create_txt_board(board_name='your-board')
```
